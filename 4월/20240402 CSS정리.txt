CSS란
웹페이지를 좀 더 보기좋은 형태로 디자인하는 방식
(통계적인 리스트 확인하여 학습하기.)

CSS의 장점
- 전반적으로 웹페이지의 가독성을 올릴 수 있다.

CSS가 등장하기 전의 상황
- 문서를 만들고 수정 및 업데이트하기가 힘들었다.

CSS는 따라서 전반적으로 디자인만을 위해 만들어진 언어라고 보면된다.

CSS를 사용시 주의할점 해당페이지를 구축을 할 때 따로 폴더를 만들어 중복을 최대한 제외한다.

STYLE에는 3가지 종류가 존재
전역 = <link rel="STYLE 명칭" href="파일 경로">
그룹 = CLASS
단일 = ID
